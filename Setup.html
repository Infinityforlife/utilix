Setup & Commands

/setup

Opens the setup menu to configure Utilix for your server.

/add [user]

Adds a user to the ticket where this command is sent.

/check

Checks if a person is reported on the bot. Usage: target_type (Discord user or server), target (Server link or User ID).

/blacklist user: [User]

Blacklists a user from opening tickets in your server.

/unblacklist user: [User]

Removes a user from the blacklist, allowing them to open tickets again.

/claim

Allows a ticket staff member to claim the currently open ticket.
